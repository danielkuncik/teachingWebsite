Create a file in the finished_assignments folder called physics_functions.py. All of the functions in this program should be run from this lesson. The first line of your file should be import math. This will allow you to use the math.sqrt() function throughout the file.

The goal of these assignments is to work on using Python as a calculator effectively. All of the functions you will create are simple applications of physics formulas. All functions should return answers rounded to two decimal places!

You can use the test commands that I wrote to help test that your file works! However, after checking that these commands, work, you should delete them! In the end, your file should have only the import math statement, then five functions defined.

$$ GPE = mgh $$ $$ \Delta x = v_i (\Delta t) + \frac{1}{2} a (\Delta t)^2 $$ $$ V = IR $$ $$ a = \frac{v_f - v_i}{\Delta t} $$ $$ KE = \frac{1}{2} m v^2 $$

  1. Create a function entitled get_gpe(mass, height).
  2. Create a function entitled get_displacement(initial_velocity, acceleration, change_in_time).
  3. Create a function entitled get_current(voltage, resistance).
  4. Create a function entitled get_acceleration(final_velocity, initial_velocity, change_in_time).
  5. Create a function entitled get_velocity_from_ke(kinetic_energy, mass)

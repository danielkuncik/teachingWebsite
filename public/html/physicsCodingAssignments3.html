New Commands

  • float(): Converts a number into a floating point value. Necessary for correctly dividing two numbers in a function, if you want an accurate decimal result
  • import math: Used as the first line of your program. Allows you to use a library of mathematical operations in your program.
  • math.sqrt(): takes the square root of a number
  • round(): Rounds a number to a specified number of decimal places. The command takes two arguments. The first argument is a number, and the second is the number of decimal places you must round it to.

It is a good idea, before completing the assignment, to test out these new functions! Go into the python console by typing python in any terminal. There, you can type any of these commands to test how they work!

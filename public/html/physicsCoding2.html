  3. Graphing

    For scientific computing, we need to make graphs. We are going to use a graphing program called matplotlib. First, we would need to install matplotlib, but fortunately that's already done for us by the Cloud9 platform. Make sure you are in the Python Console (not the UNIX operating system). The line should begin with three carats >>> And, then type import matplotlib.

    Making a very simple plot, you need to enter commands in precisely this order!

    1. import matplotlib
    2. matplotlib.use('Agg')
    3. import matplotlib.pyplot as plt A couple warnings show up, don't worry about them.
    4. a = [0,1,2,3]
    5. b = [0,2,4,6]
    6. plt.plot(a,b) A message comes up saying you have a lines.lines2D object.
    7. plt.savefig('myFirstPlot.png')
    8. Now, the file myFirstPlot.png should appear in your File System. Click on it to see your plot!

    If every time you enter and exit the Python Console, then you must retype the first three commands! But, if you don't exit the console, you don't need to worry about them!

  4. The Range Function

    The range() function on Python will help us make lists we can use to graph functions!

    range(start, stop)

    Range only takes integer values. See what you get when you type the following code:

    • range(0,12)
    • range(20,30)
    • range(-10,10)
    • range(12)
    • range(6)

    The range() function is mostly pretty intuitive, but there is one funny thing about it: range(0,12) only returns numbers from 0 to 11, and doesn't include 12!

  5. Making a (not very interesting) Graph in a File

    In order to make graphs for more complex situations, we need to run multiple commands, which is far simpler if we use a Python File, rather than just typing commands in the Python Console.

    Make sure you are in the Ubuntu operating system. If you are, the line on your terminal should end with a dollar sign $.

    touch making_a_graph.py

    This creates a new Python file called making_a_graph.py. Open it up in your File System.

    In the graph, you need to type some code that will create multiple lists and then create functions based upon them. Type this into your file:

    x = range(101)
    y = [5]*101
    print(x,y)

    Then, try running your code in the UNIX terminal by typing python making_a_graph.py. It should give you two lists. The first one (x), is a list of numbers from 0 to 100. The second one, y is a list of 101 fives. Delete the print(x,y) statement from your code, because you don't really need it.

    Let's try graphing these lists Although the result won't be very interesting, it will be good practice! At the top of your program, you need to type

    import matplotlib
    matplotlib.use('Agg')
    import matplotlib.pyplot as plt

    Then, at the bottom of your program, you need to type:

    plt.plot(x,y)
    plt.savefig('myGraph.py)

    Next, try running a program the program in your terminal by typing python making_a_graph.py into your terminal. If you did this correctly, the file myGraph.py will appear in your File System! If you open it, you'll see a graph from x = 0 to x = 100 that is always five.

    Challenge: Change this code to make a graph that is also 30 (or any number you like!). If you change the code and rerun it, then the myGraph.png file will update and you can reload it to see the newer graph.

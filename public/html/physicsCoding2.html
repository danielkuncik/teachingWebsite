  2. Introduction to Lists

    To learn about lists and arrays, we are going to use the Python Console. Open up the Python Console by entering the command python in UNIX.

    First, make a list, and show that you made a list....

    1. >>>myList = [1,2,3]
    2. >>>print(myList)
    3. >>>type(myList)

    Small List Challenge:

    1. Make a list of all the numbers from 1 to 10.
    2. Use print to confirm that you made it!

    Length of Lists: Use the function len() in order to see how many elements a list has!

    1. myList = [1,2,3,4,5,6,7,8]
    2. len(myList) # this should return the number 8!
    3. See if you can make a list with a length of 9, then use len() to confirm that you did it!

    Reading your list in pieces: You can read individual elements of a list with this syntax list[index] The index variable starts at zero! Try following the steps to practice this!

    1. myList = [2,4,6,8,10]
    2. print(myList[0])
    3. print(myList[1])
    4. print(myList[3])
    5. print(myList[10]) Huh? This one doesn't work! Why?

    Small list challenge:

    1. Make a list up to 30 of all numbers, counting by 3s.
    2. Use len to confirm the list has three elements.
    3. Check what element 0 is.
    4. Check what element 3 is.
    5. The sixth item in your list should be 18. But if I type myList[6], I get 21, not 18! What do I need to type to get the 6th element, 18? Why?

    Changing Parts of Your List. You can change each individual element of a list with indexes, similar to how you read each element of a list.

    1. myList = [1,2,3,4,5,6]
    2. print
    3. myList[0] = 0
    4. myList[1] = 0
    5. print(myList) What happened, what changed?

  3. Graphing

    For scientific computing, we need to make graphs. We are going to use a graphing program called matplotlib. First, we would need to install matplotlib, but fortunately that's already done for us by the Cloud9 platform. Make sure you are in the Python Console (not the UNIX operating system). The line should begin with three carats >>> And, then type import matplotlib.

    Making a very simple plot, you need to enter commands in precisely this order!

    1. import matplotlib
    2. matplotlib.use('Agg')
    3. import matplotlib.pyplot as plt A couple warnings show up, don't worry about them.
    4. a = [0,1,2,3]
    5. b = [0,2,4,6]
    6. plt.plot(a,b) A message comes up saying you have a lines.lines2D object.
    7. plt.savefig('myFirstPlot.png')
    8. Now, the file myFirstPlot.png should appear in your File System. Click on it to see your plot!

    If every time you enter and exit the Python Console, then you must retype the first three commands! But, if you don't exit the console, you don't need to worry about them!

    Try making a new list:

    1. c = [0,1,2,3,4,5]
    2. d = [0,1,4,9,16,25]
    3. plt.plot(c,d)
    4. plt.savefig('mySecondPlot.png')

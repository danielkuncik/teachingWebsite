Three new Python Calculator Commands

In order to effectively use python as a calculator, you need to understand the following three commands.

Integers vs. Floats

In python, integer values and floating point values are treated differently. An integer is just like an integer in math class. Examples of integers are:
5
-8
100
-90
A floating point value is like a decimal number in math class. Examples of floating point values are:
8.0
6.7231
-10.0
-6.7821
Floating point values and integer values are different types of computer variables that will be treated differently in python! The most obvious differnce comes when you try to divide two values.

Kinematics Project: Automatic Kinematic Graphs

Requirements:

Create a class entitled kinematicsSituation. The kinematicsSituation class should have three initial variables: initialPosition, initialVelocity, and acceleration. These three variables represent the three constant values within any one dimensional kinematics situation. The variables initialPosition and initialVelocity initial variables are numbers representing the position and velocity of the object at time = 0. The acceleration initial variable is a number that represents the acceleration, which is constant in the kinematics situation we are considering.

The kinematicsSituation class should have three methods, drawPositionTimeGraph, drawVelocityTimeGraph, and drawAccelerationTimeGraph. These three methods should each draw a the selected kinematic graph and output it onto a .png file. Each method should take three arguments timeMin, timeMax, and filename. The arguments timeMin and timeMax are numbers representing the range of time that will be drawn in the position, velocity, and acceleration graphs. The filename argument is a string that gives the name of the file onto which the graph will be outputted. The output graphs must be well drawn and contain the correct labels on x-axis and y-axis.

Note that this problem deals with only one-dimensional situations with a constant acceleration. (Extensions into multi-dimensional or stepwise acceleration will be included in future projects.)

Applications of the Project:

Use the Python Class you have created to create kinematic graphs representing common situations.

  • A car accelerates at a rate of 3 m/s2 from rest for a time of 5 seconds. Draw position, velocity, and acceleration graphs for the car.
  • You drop a rock off of a roof from rest and it strikes the ground 2 seconds later. Create position, velocity, and acceleration graphs for the rock. [The initial height of the rock is 19.6 meters.]
  • A car moves at a constant velocity of 18 m/s for a time of 8 seconds. Create position, velocity, and acceleration graphs for the car.
  • A person sits at the origin and does nothing for 10 seconds. Create position, velocity, and acceleration graphs for the person.
  • A car beginning from a position 50 m to the LEFT of the origin approaches with a speed of 15 m/s. It comes to rest at the origin 5 seconds later. Create position, velocity, and acceleration graphs for the person.

Note that the goal is only to create a class which meets the requirements above, and it is okay to test each of these applications from within your code. You do not need to create an API (application program interface) to go with the Python class you created. (If you are interested in web design, that might be a good side project, but for a physics class we are going to focus primarily on solving the problems.)

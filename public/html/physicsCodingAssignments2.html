  2. You should already have a program in the finished_assignments folder called simple_python_functions.py. There should be one function in this program called get_y(m,x,b). Create a second function called speed_of_light(). This function takes no arguments, and it always returns the speed of light.
    The first line of this function should look like:
    def speed_of_light():
    In order to test that you have correctly run your function, if you add the line print(speed_of_light()), it should return the value of the speed of light.
  3. Make another function in the simple_python_functions.py program called add_then_subtract(a,b,c). This function should return a added to b then subtracted by c.
  4. Create a new program called simple_physics_formulas. In this program, create a series of functions that replicate the math in common physics formulas. You can find a good list of common physics formulas here. This program should include the following functions:
    • get_momentum(mass, velocity)
    • get_kinetic_energy(mass, velocity)
    • get_frequency_of_electromagnetic_wave(wavelength_in_meters)
    • get_energy_of_electromagnetic_wave(frequency_in_hertz)

    To test this idea, you can add the following lines below the functions. Please delete them after you have tested your file:

    print(get_momentum(5,6)) # should return 30
    print(get_momentum(3.86,9.12)) # should return 35.203199999999995
    print(get_kinetic_energy(1,2)) # should return 2.0
    print(get_kinetic_energy(4,5)) # should return 50.0
    print(get_kinetic_energy(6.89,10.23)) # should return 360.5292405
    print(get_frequency_of_an_electromagnetic_wave(5.56e-7)) # should return 539568345323741.06 (or a similar value)
    print(get_frequency_of_an_electromagnetic_wave(4.56e-9)) # should return 6.578947368421053e+16 (or a similar value)
    print(get_frequency_of_an_electromagnetic_wave(1.23e-4)) # should return 2439024390243.9023
    print(get_energy_of_an_electromagnetic_wave(5.55e14)) # should return 3.67743e-19
    print(get_energy_of_an_electromagnetic_wave(3.12e12)) # should return 2.067312e-21
    print(get_energy_of_an_electromagnetic_wave(8.54e16)) # should return 5.658604e-17

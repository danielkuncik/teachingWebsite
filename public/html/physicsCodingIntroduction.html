Why are we coding in physics?

Because real scientists don't just solve problems on pen and paper! Almost all scientists these days are spending a considerable portion of time writing computer code. Almost all the problems that people would want to sovle today require it. For example, we've solved many problems that involve a car moving at a constant acceleration. Imagine now you have a rocket ship taking off. Is its acceleration constant? Certainly not. How many different equations and calcualtions do you think you need to explain the motion of a rocket taking off? Thousands, if not millions or billions! Humans can't do that many equations at once, but what can? A computer! That's why real scientists today do so many of their calculations by writing a computer program.

  3. Very Basic Unix BASH Commands

    Computer programmers frequently use an operating system called Unix. Your Cloud9 computer, which you are currently accessing, is a unix computer. Unlike operating systems like Windows or MacOSX, you need to use typed commands to work with a unix system. In this case, you enter Unix commands and reiceve the output in the terminal area of the Cloud 9 interface. The type of UNIX you are using is also called BASH. Try out each of the following commands!

    • pwd: "Print Working Directory" prints what directory you are currently.
    • ls: "List" lists everything in the directory you are currently in.
    • mkdir: "Make Directory" Makes a new directory. A directory is just like a file in Windows or MacOSX.
    • cd: "Change directory" Changes what directory you are working in.
    • cd .. : This changes to the directory one above where you are in the file tree.
    • cd ~ : this returns to your home directory (where you started).
    • rm: "Remove" gets rid of something
    • clear: clears your terminal screen

    See if you can use these commands to do the following!

    1. Make a new directory called "physics_problems."
    2. Use "ls" to confirm that you made it. It should be in the list.
    3. Go into that directory.
    4. Use pwd to confirm you are in the physics_problems directory.
    5. use "cd .." to go back.
    6. Keep using "cd .." to go back farther into the root directory. Every time you use it, use "pwd" and "ls" to see what's going on!
    7. Return to your home directory.
    8. Delete the "phyiscs_problems" directory.
    9. Use "ls" to confirm that it's gone.
    10. Use "clear" so that you have a nice, clear terminal screen.

    If you have any interest at all in any career that involves coding and program, it behooves you to become as comfortable as possible with unix commands.

  4. Running The Python Console and generating a "Hello World" program.

    We will be coding using a programming language called Python. Python is the most commonly used langauge for scientific computing! Scientific computing just means solving scientific problems by computing, which is what we are going to do.

    To begin running Python, we are going to use the Python Console. The console allows you to enter commands and immediately get a result, as opposed to the more common method of coding in which you write several lines of code (we are going to do that also!).

    To start running the Python console, type python into your terminal and click enter. You will know you are inside because you see a >>> .

    Before doing anything else, know how to exit the Python Console, so you don't get trapped inside! To exit, just type
    exit()
    Before doing anything else, exit and reenter. That way, you know you're in control of where you are!
